## 六、常见问题排查

### 6.1 Neo4j连接失败

```
【系统提示】Neo4j图谱未连接，正在使用本地数据库查询。
```

**排查步骤：**
1. 检查Docker是否运行: `docker ps`
2. 检查Neo4j是否启动: `docker logs neo4j`
3. 检查配置文件: `configs/neo4j_config.yaml`
4. 测试连接: `curl http://localhost:7474`

### 6.2 数据导入失败

```
✗ SQLite数据库不存在
```

**解决方案：**
```bash
python scripts/init_sqlite.py
python scripts/import_neo4j.py
```

### 6.3 冲突检测未触发

**检查项：**
1. `parent_constraints` 是否正确填充
2. `student_preferences` 是否正确填充
3. 分数/位次是否在冲突触发阈值内
